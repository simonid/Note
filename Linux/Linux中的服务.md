# Linux的服务

##RPM系
分类:<br>
```
Linux服务：
(1)RPM包默认安装的服务
    1.独立的服务
    2.基于xinetd服务
(2)源码包安装
```
RPM包安装的服务：
```
chkconfig  --list  //查看服务自启动状态，可以看到所有rpm报安装的服务
```
源码包安装的服务：<br>
查看服务安装位置，一般是`/usr/local`下<br>
RPM安装服务和源码包安装服务的区别就是安装位置不同<br>
源码包安装在指定位置，一般是`/usr/local`下<br>
RPM包安装在默认位置<br>
由于安装位置不同，所以他们的管理方式也不同<br>
